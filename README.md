# DebugMenu

This is a very simple debug menu for Road to Vostok with:

- God mode
- Infinite stamina
- No ailments
- NPC ignores player (This is broken, WIP!)
- No weight limit (big weight limit, actually)
- Running speed multiplier
- Jump height multiplier
- Unstick (moves you up 2 units)

Press **F4** to toggle the panel (shows cursor while open, re-captures on close).

## Known issues

- F4 keeps camera moving, didn't look into this yet
- NPC ignores player just literally doesn't work lol

## Install

1. Install <https://modworkshop.net/mod/55623>
2. Add `DebugMenu.vmz` to your `mods` folder

## Building

`zip -r DebugMenu.vmz mod.txt mods/`
